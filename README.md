# Himani's Portfolio

Personal portfolio application built with TanStack Start, React, TypeScript, and Tailwind CSS. The codebase is organized as a developer-owned frontend project, with clear route structure and room for future backend integrations.

## Tech Stack

- TanStack Start
- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- Lucide React
- Motion

## Environment Variables

No environment variables are required for the current frontend-only version of the project.

If backend services are added later, document new variables here and keep server-only values out of client-side route code.

## Backend Integration Notes

The current project is frontend-first. A future backend can be introduced gradually without restructuring the app:

- Replace hardcoded portfolio content in `src/lib/projects.ts` with API-backed project data.
- Move contact and availability data into a CMS, database, or profile service if it needs to be updated outside the codebase.
- Add server functions or API routes for contact form submission, resume delivery rules, analytics, or authenticated admin editing.
- Keep route components focused on presentation and data shaping so backend integration stays localized.

## Notes for Future Development

- `src/routeTree.gen.ts` is generated by TanStack Start and should not be edited by hand.
- The current content includes placeholder portfolio details that should be personalized before launch.
- The current workspace folder name may still reflect the original download directory; repository metadata and source references have been cleaned, but the physical folder rename should be done outside the running workspace if desired.
